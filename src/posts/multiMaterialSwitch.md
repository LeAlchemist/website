The Multi Material Switch node is a powerful custom node that allows the user to switch between 20 full materials with the option to use any of the 12 texture map types (Diffuse, Base Color, Normal, Specular, Emissive, Glossiness, Roughness, Metallic, Specular Level, Ambient Occlusion, Height and Opacity) to fit almost all texture/render pipe lines the user may have.
