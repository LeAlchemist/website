This was a collection of fantasy based sticker designs that I did based off some old sketches. I had doodled these on stream to be used as both an on stream asset, and to sell on Redbubble. This was a fun cute collection of stickers to work on, and helped me work on a Substance Designer node for creating sticker outlines and sticker like effects.
